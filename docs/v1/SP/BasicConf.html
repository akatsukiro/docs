# 关闭注册

此功能默认关闭，开启后即关闭注册，注册按钮会呈现不可点击的状态。

# 网站标题

网站的标题。

# 网站关键字

网站关键字，利于搜索引擎关键字索引，多个请使用英文逗号分隔。

# 自定义CSS

使用自定义 css 可以做到覆盖整站的任何标签样式，保存前建议先压缩 css

# 统计代码：

可自定义第三方统计代码

# 备案号

域名备案号，填写后会显示在网站底部。

# 系统公告

将会在首页显示的公告内容，为空则不显示，支持使用 html 标签
