# 原图保护

提示

原图保护的作用是隐藏图片的真实 url 路径，未开启前图片是通过运行环境输出并缓存的，开启后请求该图片会通过 PHP 接管，由 PHP 处理图片的输出。

警告

原图保护功能对服务器配置要求较高，虽然有缓存驱动，但默认情况下使用的是文件缓存，原理还是读取本地的文件，占用 IO 读写，且每次都将图片内容读取到内存中，对服务器内存有着比较高的要求。 这种负载压力在图片请求量(除了缩略图)过多的时候尤为明显。
